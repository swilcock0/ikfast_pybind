.. Write project description

**ikfast_pybind** is a python binding generation library for the analytic kinematics engine `IKfast <http://openrave.org/docs/1.8.2/openravepy/ikfast/). The python bindings are generated via [pybind11](https://github.com/pybind/pybind11>`_ a `CMake <https://cmake.org/>`_-based build system.
